# Ecological-Footprint-Analysis
Ecological Footprint Analysis | Python, Pandas, NumPy, Scikit-learn, Matplotlib

Performed data cleaning and preprocessing on global ecological footprint dataset by handling missing values and converting socio-economic indicators to numeric formats.

Conducted exploratory data analysis (EDA) and visualized relationships between footprint factors such as carbon, cropland, and forest usage.

Implemented Linear Regression model to analyze and predict Total Ecological Footprint based on indicators like GDP, HDI, and Life Expectancy.

Generated statistical insights and trend visualizations across countries, income groups, and regions.

Built reusable data transformation workflow for structured environmental data analysis.
